Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h2 element acts as a second heading of the page.
  • Meta has information that is used by the browser.
  • The base element establishes the base URL for a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declerations also contain two important things being: the CSS property we want to apply and the value of that property.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • A link element creates the connection to the external file.
  • In the head section is where metadata is contained.
  • When you would want to style an elemnt directly in HTML, you use inline CSS.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is a sset of instructions that tells the computer how to preform a task in a certain way.
  • The name of a function is followed by paranthesis where inside it, the parameters will be defined.
